## Red Flags for Communication Challenges

From course learnings, these regulatory characteristics signal communication difficulty:

- [ ] Unclear enforcement mechanism → Sellers don't know who's checking
- [ ] Ambiguous compliance definition → Different teams interpret differently
- [ ] High implementation cost → Business resistance despite clear requirements
- [ ] Cascading requirements → One compliance step triggers others
- [ ] Prerequisite knowledge gaps → Can't understand without context
- [ ] Conflicting standards → Different frameworks/interpretations
- [ ] Rapid regulatory change → Moving target
- [ ] New precedents → No historical implementation examples

**Framework application**: Compliance_tracker should flag these in alerts, message_optimizer should prepare proactive communication.
